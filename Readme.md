
# HUS-EXPRESS-DEMO

A simple ExpressJS server.


## Authors

- [quangtv@rabiloo.com](quangtv@rabiloo.com)

## Installation

```bash
  npm install
```
## Running
- Production mode:
```bash
  npm run start
```

- Dev mode:
```bash
  npm run dev
```


## API Reference

| Endpoint          |  Description                          |
| :-----------------|  :------------------------------------|
| `/account`        |  Account of student                   |
| `/student`        |  Student information                  |
| `/class`          |  Class student study                  |
